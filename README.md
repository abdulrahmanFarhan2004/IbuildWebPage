# الهوية البصرية لموقع IBuild — The Java Developer Club

## 1) الشعار والكلمة
- الكلمة الرئيسية: **IBuild.java** (مستعملة في الشريط العلوي والهوامش).
- أيقونة المربع: مربع بحواف دائرية (9px تقريباً) مع تدرّج ثلاثي الألوان (البرتقالي → الوردي → التركوازي)، نص داخلي بحروف **IB** بخط JetBrains Mono عريض أبيض.
- الاستخدام: يجاور الشعار عادة نص "IBuild.java" بتدرج حيّ (Gradient Animation) متحرك كل 4–5 ثوانٍ.

## 2) لوحة الألوان
- البرتقالي الأساسي `#FF6B35` (jo)
- برتقاني فاتح `#FF8C42` (jo2)
- برتقاني ذهبي `#FFB347` (jo3)
- تركوازي `#00D4AA` (teal)
- أزرق فاتح `#4FC3F7` (blue)
- بنفسجي `#9B59FF` (pur)
- وردي `#FF3CAC` (pink)
- خلفية داكنة `#060A0F` (bg)
- سطح داكن `#1A2332` (surf)
- نص ثانوي `#94A3B8` (tm) — نص رئيسي أبيض `#FFFFFF` (tp)
- حدود/زجاجية شفافة: rgba(255,255,255,0.05–0.10)
- التدرج الرئيس: `linear-gradient(120deg, #FF6B35, #FF3CAC, #9B59FF, #00D4AA)`

## 3) الخطوط والتدرج الهرمي
- العناوين الرئيسية: **Cinzel** (وزن 700–900، حجم يصل إلى 5.4rem في البطل Hero).
- العناوين الثانوية والأزرار: **JetBrains Mono** (أحرف مونو، وزن 700–800).
- النصوص المتنية: **Inter** أو **Tajawal** (للعربية)، وزن 400–700، حجم 0.9–1.1rem.
- الحروف الكبيرة في الشرائح/الشارات: JetBrains Mono بحروف متباعدة (letter-spacing حتى 0.14em).

## 4) الأسلوب العام (Style)
- خلفية داكنة مع ضبابية زجاجية (Glassmorphism) وحدود رفيعة شفافة.
- ظلال ناعمة وعميقة: `box-shadow: 0 8px 32px rgba(0,0,0,0.45)` ويزداد عند التحويم.
- استخدام بقع ضوئية ضبابية (Ambient blobs) بألوان دافئة/باردة متحركة ببطء.
- شبكة خفيفة في خلفية البطل (Hero) مع قناع شعاعي ناعم.
- بطاقات بحد علوي متدرج يظهر عند التحويم.

## 5) الحركة والميكرو-تفاعلات
- تحريك التدرجات (gs) كل 4–5 ثوانٍ.
- تأثير Fade-Up عند الظهور (Intersection Observer).
- عناصر تطفو (cfloat) في البطل.
- نبضة زر/شارة (pulse) لنقاط الحالة.
- مؤشر مخصص: نقطة داخلية + حلقة خارجية تكبر عند التحويم على روابط/أزرار/بطاقات.

## 6) المكونات الرئيسية
- **Nav Bar**: شريط علوي زجاجي دائري الزوايا، شعار يسار/يمين (حسب RTL)، روابط شفافة تتحول لخلفية زجاجية عند التحويم، زر CTA متدرج.
- **Hero**: عنوان مركزي متعدد الأسطر، عبارة فرعية، زران (Primary متدرج، Ghost زجاجي)، كتل كود عائمة بنمط JetBrains Mono، شبكة خلفية وتوهجات.
- **Marquee**: شريط تمرير أفقي يعرض تخصصات (Java SE, Spring Boot, DSA ...)، بنقاط فاصلة وردية.
- **Sections**: عناوين Cinzel مع إبراز متدرج، فقرات نصية بعرض 520–620px، هوامش رأسية واسعة (حوالي 110px).
- **Badges/Chips**: شارات بحروف متباعدة وخلفيات شبه شفافة (cyan/orange/green/purple) مع حدود خفيفة.
- **Cards**: خلفية زجاجية، حافة متدرج علوي عند التحويم، حشوة 26–34px، زوايا 16–20px، ظلال عميقة.
- **Stats**: شبكة 2×2 بأرقام متدرجة كبيرة (2.6rem) وخط JetBrains Mono.
- **Activities Grid**: شبكة 3 أعمدة (تتحول لعمود واحد تحت 960px)، أيقونة ضمن مربع ملون، عنوان Cinzel، نص ثانوي tm.
- **Projects Grid**: بطاقات مع حالة (مكتمل/نشط/قيد البناء)، لون توهج خاص بكل بطاقة عبر CSS variable `--proj-color`.
- **Video/Lessons Cards**: صورة غلاف 16:9، زر تشغيل دائري برتقالي، شريط متدرج أسفل البطاقة يظهر عند التحويم، عنوان Cinzel 0.9rem.
- **Team**: بطاقات قيادة كبيرة بوهج ملون ودائرة خلفية ضبابية؛ بطاقات أعضاء صغيرة شبكية.
- **Join CTA**: حاوية زجاجية كبيرة بحواف 28px مع بقع ضوئية خلفية، عنوان Cinzel، زران (Primary متدرج، Ghost زجاجي).
- **Footer**: شبكة 4 أعمدة، حدود متدرجة، شارات صغيرة بزجاجية، نسخة حقوق بلغة JetBrains Mono.

## 7) النِسَب والاستجابات (Responsive)
- نقطة كسر رئيسية عند 960px: إخفاء روابط الناف، تحويل الشبكات (أنشطة/مشاريع/فريق) إلى عمود واحد، تقليل الهوامش الأفقية، إخفاء كتل الكود العائمة.
- حواف البطاقات 14–24px، حشوات الأقسام 72–110px حسب العرض.

## 8) نبرة العلامة (Voice & Tone)
- عربية/إنجليزية ثنائية، ودودة وتشجيعية لمجتمع مطوري Java.
- تركيز على العمل الجماعي، التعلم المستمر، وبناء مشاريع حقيقية.
- استخدام عبارات مختصرة وواضحة مع لمسة تقنية (مثل التعليقات // في الشرائح).

## 9) الأصول المقترحة للاستخدام
- شعار متجه (SVG) بمربع متدرج ونص JetBrains Mono أبيض.
- تدرج أساسي جاهز للاستخدام في الأزرار والخطوط البارزة.
- نمط خلفية: شبكة خفيفة + بقع ضوئية ضبابية (يمكن تصديرها كصورة خلفية ثابتة أو إبقائها CSS).

## 10) الاختصارات المتكررة في CSS
- متغيرات الألوان في :root كما في `index.html`.
- تأثير الزجاجية: `background: rgba(255,255,255,0.05-0.10); border: 1px solid rgba(255,255,255,0.09); backdrop-filter: blur(18-40px);`
- الظلال: `box-shadow: 0 8px 32px rgba(0,0,0,0.45)` مع تعزيز عند التحويم.
- الحركة: `animation: gs 4s linear infinite` للتدرج، `fadeUp .8s` للظهور، `cfloat 8–11s` للتطفو، `pulse 2s` للنقاط.
